This is a visualization dashboard with both ends for the [UNB CIC](http://unb.ca/cic/).

Javascript for frontend, and Python for the backend.

Libraries include:
* **bootstrap** (layout)
* **d3** (visualization)
* **jquery** (DOM)
* **datatables** (data display)

and
* **Flask** (web server)

Package management is **bower**.
