Social Hub is a rules engine with both front end and back end.

Javascript for the frontend, and Java for the backend.

Libraries include:
* **Spring** framework:
  * **Spring-MVC**
  * **Spring-Security**
  * **Spring-Integration**
* **Freemarker** and **JSP**

and
* **JQuery** and **JQuery-UI**
